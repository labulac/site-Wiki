### 问题来源

Mac 升级系统后发现有些非官方渠道的软件，无法安装

同时发现 **安全性与隐私** 的 **允许从以下位置下载的应用：** 中的**任何来源**选项也不见了...

### 解决办法

在终端中输入以下命令：

```bash
sudo spctl --master-disable
```

然后在设置里又可以看到<u>允许未知来源</u>又出现了

![](https://cdn.jsdelivr.net/gh/labulac/pic@master/uPic/aQdxf1.png)

点击那个小黄锁，输入密码，选中。

然后安装你想要安装的软件 ！
